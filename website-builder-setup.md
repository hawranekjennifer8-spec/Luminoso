## 🚀 Erste Schritte

### Schritt 1: Shopify-Account vorbereiten

1. Öffne deinen Shopify Admin: https://admin.shopify.com
2. Gehe zu **Online Store** → **Themes**
3. Stelle sicher, dass das **Luminoso Theme** mit GitHub verbunden ist
4. Der Status sollte "Connected to GitHub" zeigen

### Schritt 2: Theme aktivieren

1. Wähle das **Luminoso Theme**
2. Klicke **Actions** → **Publish**
3. Warte auf die Bestätigung "Theme published"

### Schritt 3: Theme-Vorschau

1. Nach der Veröffentlichung klicke auf **Actions** → **Preview**
2. Die Website sollte vollständig geladen werden
3. Falls nicht, siehe [Troubleshooting](#troubleshooting)

## 🔗 GitHub-Integration

### Repository verbinden

Die Luminoso-Website ist bereits mit GitHub verbunden:

- **Repository:** `hawranekjennifer8-spec/Luminoso`
- **Branch:** `main`
- **Aktualisierungen:** Automatisch wenn du änderungen pusht

### Theme synchronisieren

Wenn Shopify dein GitHub-Theme nicht automatisch aktualisiert:

1. Gehe zu **Online Store** → **Themes**
2. Wähle **Luminoso**
3. Klicke **Actions** → **Reload theme**
4. Warte 2-3 Minuten

## 🎯 Theme-Aktivierung

### Live schalten

1. **Online Store** → **Themes**

- `your-shop-name.myshopify.com`

## ⚙️ Grundeinstellungen

### Logo hochladen

1. **Online Store** → **Themes** → **Luminoso** → **Customize**
2. Klicke **Theme settings** (unten links)
3. Gehe zu **Branding** → **Logo**
4. Lade dein Logo hoch (empfohlen: PNG 500x200px)
5. Klicke **Save**

### Farben anpassen

1. In **Theme settings** → **Farben**
2. Passe folgende Farben an:
   - **Primary White:** #F9F8F6 (Hintergrund)
   - **Primary Black:** #111111 (Text)
   - **Accent Gold:** #D4AF37 (Buttons, Highlights)
   - **Champagne Gold:** #F0E68C (Sekundär-Accents)

### Typografie ändern

1. **Theme settings** → **Typografie**
2. **Heading Font:** Cormorant Garamond oder deine Wahl
3. **Body Font:** Inter oder deine Wahl

## 📦 Produktverwaltung

### Produkte hinzufügen

1. Gehe zu **Products** im Shopify Admin
2. Klicke **Add product**
3. Fülle folgende Felder aus:
   - **Title:** z.B. "Brillant-Ring Gold"
   - **Description:** Detaillierte Produktbeschreibung
   - **Images:** Hochqualitäts-Fotos (min. 1000x1000px)
   - **Price:** Verkaufspreis
   - **Compare at price:** Originalpreis (optional, für Sales)
   - **SKU:** Eindeutige Artikel-Nummer

### Kategorien erstellen

1. **Products** → **Collections**
2. Erstelle Sammlungen für:
   - **Necklaces** (Halsketten)
   - **Bracelets** (Armbänder)
   - **Rings** (Ringe)
   - **Earrings** (Ohrringe)
   - **Best Sellers** (Bestseller)

### Produkte zu Kategorien hinzufügen

1. Öffne eine Kollektion (z.B. "Rings")
2. Klicke **Add products**
3. Wähle Produkte aus und speichere

## 🏠 Homepage gestalten

### Hero-Sektion anpassen

1. **Online Store** → **Themes** → **Customize**
2. Klicke auf den **Hero** oben auf der Seite
3. Bearbeite:
   - **Eyebrow Text:** "Exklusive Kreationen"
   - **Heading:** "Timeless Elegance"
   - **Subtitle:** Deine Beschreibung
   - **CTA Text:** "Zur Kollektion"
   - **CTA Link:** `/collections/all`
   - **Background Image:** Lade ein Premium-Bild hoch

### 3D-Ring-Animation einbinden (optional)

1. Erstelle eine 3D-Szene auf **Spline.design**
2. Kopiere die Spline Scene URL
3. Im Theme Editor → **Hero** → **Spline Scene URL** einfügen

### Weitere Sections anpassen

- **All Products:** Zeigt alle Produkte
- **Shop by Category:** Kategorien mit Bildern
- **Customer Reviews:** Kundenbewertungen
- **Trust Section:** Vertrauens-Signale
- **Brand Story:** Deine Markengeschichte

## ⚖️ Rechtliche Seiten

### Impressum erstellen / bearbeiten

1. **Online Store** → **Pages**
2. Klicke auf **Impressum** (oder erstelle neue Seite)
3. Fülle folgende Daten aus:
   - Dein Name / Firmenname
   - Adresse
   - E-Mail
   - Telefon
   - Umsatzsteuer-ID
   - Handelsregister-Nummer

### Datenschutzerklärung

1. Öffne die **Datenschutzerklärung** Seite
2. Aktualisiere:
   - Verantwortliche Stelle (dein Name/Unternehmen)
   - Datenschutzbeauftragte/r (optional)
   - Speicherdauer
   - Externe Dienstleister

### AGB und Widerrufsrecht

1. Öffne **AGB** und **Widerrufsbelehrung**
2. Prüfe auf Aktualität (besonders für Deutschland)
3. Aktualisiere Versandkosten, Lieferfrist, etc.

## 🎨 Design anpassen

### Farben ändern

1. **Customize** → **Theme settings** → **Farben**
2. Wechsel zwischen Luxus-Farbschema oder deine Farben

### Schriftarten wechseln

1. **Theme settings** → **Typografie**
2. Wähle aus Shopify-Fonts oder Custom Fonts
3. Empfohlen: Serif für Headlines, Sans-Serif für Body

### Layout anpassen

1. **Customize** → Klicke auf Sections
2. Bearbeite Spacing, Grid, Größen
3. Passe für Mobile an (unter 768px)

### Animationen deaktivieren

1. Im **assets/css/animations.css** oder Theme Editor
2. Entferne/kommentiere Animation-Klassen
3. Oder nutze CSS `animation: none;`

## 🔍 Suchmaschinen-Optimierung (SEO)

### Meta-Beschreibungen

1. Für jede **Page** und **Product**:
2. Gehe zu **SEO** Section
3. Schreibe Meta-Description (max. 160 Zeichen)
4. Beispiel: "Premium Schmuck handgefertigt. Luxus-Ringe, Halsketten & Armbänder bei Luminoso."

### Sitemap generieren

1. Shopify generiert automatisch eine Sitemap
2. Zu finden unter: `yoursite.com/sitemap.xml`
3. Diese wird automatisch bei Google indexiert

### Google Search Console

1. Gehe zu **Google Search Console**
2. Verbinde deine Domain
3. Füge die Sitemap hinzu: `/sitemap.xml`
4. Überwache Rankings und Fehler

### Strukturierte Daten (Schema Markup)

Das Theme enthält bereits:
- ProductSchema für Produkte
- OrganizationSchema für dein Unternehmen
- BreadcrumbList für Navigation

Diese werden automatisch vom Theme hinzugefügt.

## 🚀 Live-Schaltung

### Pre-Launch-Checklist

- [ ] Alle Produkte hochgeladen + Bilder
- [ ] Kategorien erstellt und gefüllt
- [ ] Logo hochgeladen
- [ ] Farben und Schriftarten konfiguriert
- [ ] Rechtliche Seiten ausgefüllt
- [ ] Impressum mit deinen Daten
- [ ] Datenschutz aktualisiert
- [ ] AGB aktualisiert
- [ ] Widerrufsrecht eingefügt
- [ ] Header und Footer überprüft
- [ ] Mobile-Ansicht getestet
- [ ] Links überprüft (keine 404 Fehler)
- [ ] Email-Adresse für Support konfiguriert

### Go Live

1. **Theme aktivieren:** Actions → Publish
2. **Shop-Passwort entfernen:** Settings → Online Store → Password
3. **Domain einrichten:**
   - Standard: `yourshop.myshopify.com`
   - Custom: Verlinke deine eigene Domain unter **Settings** → **Domains**
4. **SSL-Zertifikat aktivieren:** Kostenlos in Shopify enthalten
5. **Erste Tests durchführen:**
   - Desktop-Ansicht
   - Mobile-Ansicht
   - Tablet-Ansicht
   - Produkte kaufen (Test-Bestellung)
   - Links klicken

### Post-Launch

1. **Google Analytics aktivieren:**
   - Gehe zu **Admin** → **Sales channels** → **Online store**
   - Füge Google Analytics ID ein

2. **Facebook Pixel (optional):**
   - Admin → Settings → Customer events
   - Füge deine Facebook Pixel ID ein

3. **Newsletter einrichten:**
   - Shopify hat automatisch einen Newsletter-Builder
   - Customize → Footer → Newsletter Form

4. **Social Media verlinken:**
   - Settings → Online store
   - Füge Facebook, Instagram, etc. URLs ein

## 🐛 Troubleshooting

### "Not Found" statt Website

**Lösung:**
1. Refresh: Ctrl + Shift + Delete
2. Incognito-Modus prüfen
3. Cache leeren: **Theme** → **Actions** → **Reload theme**
4. Warten: Manchmal 2-3 Minuten

### Produkte zeigen nicht

1. Prüfe, dass Produkte in **Products** hinzugefügt sind
2. Prüfe Verfügbarkeit (Stock > 0)
3. Prüfe, dass Bilder hochgeladen sind
4. Prüfe Preise

### Design sieht komisch aus

1. Lade die Seite neu: F5
2. Cache leeren: Incognito-Modus
3. Prüfe CSS: **Theme** → **Edit code** → `assets/css/`
4. Prüfe Theme Settings

### Fehler im Checkout

1. Checkout ist Shopify-Standard, nicht dein Theme
2. Prüfe **Settings** → **Checkout**
3. Prüfe Zahlungsmethoden unter **Settings** → **Payment providers**
